
#  TP Calcul d'Impôts Locaux - Application Android

##  Description
Application Android développée en Java permettant de calculer le montant total des impôts locaux d'une propriété en fonction de sa surface, du nombre de pièces et de la présence d'une piscine.

##  Objectifs du TP
- Créer une interface utilisateur avec différents composants Android
- Gérer la saisie utilisateur (champs texte, cases à cocher)
- Implémenter des calculs basés sur des critères multiples
- Afficher dynamiquement les résultats
- Comprendre le cycle de vie d'une activité Android

##  Technologies utilisées
- **Langage** : Java
- **SDK Android** : API 24 (Android 7.0 Nougat) minimum
- **IDE** : Android Studio
- **Gradle** : Groovy DSL

##  Installation et exécution

### Prérequis
- Android Studio installé
- SDK Android API 24+
- Émulateur Android ou appareil physique

### Étapes de lancement
1. **Ouvrir le projet** dans Android Studio
2. **Synchroniser** Gradle : `File → Sync Project with Gradle Files`
3. **Choisir la cible** : Émulateur ou appareil connecté
4. **Lancer l'application** : Bouton  Run
